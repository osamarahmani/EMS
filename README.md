# Employee Management System

A production-style full-stack employee management and work tracking system built with React, TypeScript, Vite, Tailwind CSS, Express, and Node.js.

## Structure

- client: Vite + React + TypeScript + Tailwind UI
- server: Express API with sample dashboard endpoints

## Run locally

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm run dev
```

The app is ready for further expansion into authentication, CRUD modules, notifications, reports, and Prisma/PostgreSQL integration.
